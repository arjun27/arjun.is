# Welcome to Blot!

If you look inside your Dropbox you'll notice a new folder:

Dropbox &rarr; Apps &rarr; Blot

The Blot folder contains *welcome.txt*. That's this post! Feel free to edit *welcome.txt* and reload this page. You'll see the changes you make update instantly. Blot will publish any text file inside this folder.

To publish a new blog post, just save it as a text file and move it to this folder.

## Formatting your blog posts

Blot supports Markdown so you can add lists and headings and more to your blog posts.
